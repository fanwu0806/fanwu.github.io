# MIR3D for Autonomous Robotic Mushroom Harvesting

MIR3D is a monocular infrared geometry estimation framework for perception-driven robotic mushroom harvesting in dark environments.

## Overview
This project focuses on geometry-aware perception for autonomous mushroom harvesting under low-light or no-light cultivation conditions.

## Method
- Monocular infrared image acquisition
- Geometry estimation
- Grasp point prediction
- Motion planning for UR5e robotic harvesting
